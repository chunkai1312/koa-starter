# koa-starter [![JavaScript Style Guide][standardjs-image]][standardjs-url]

> A seed project for building web applications using Koa with MongoDB

## Usage

### Setup

```
$ npm install
```

### Developing

```
$ npm run dev
```

### Running

```
$ npm start
```

### Testing

```
$ npm test
```
